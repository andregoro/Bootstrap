Bootstrap é o mais popular framework HTML, CSS, e JS para desenvolvimento de projetos responsivo e focado para dispositivos móveis na web.

Baixar Bootstrap

Atualmente v3.3.7

Projetado para qualquer um, em qualquer lugar.

Bootstrap torna o desenvolvimento front-end web mais rápido e fácil. Ele é feito para pessoas de todos os níveis e dispositivos de qualquer forma ou tamanho.

Pré-processadores

Bootstrap usa CSS tradicional, mas seu código fonte utiliza os dois pré-processadores CSS mais populares, Less e Sass. Comece a desenvolver rapidamente com seu CSS pré-compilado ou construa com seu código fonte.

Um framework, qualquer dispositivo.

Fácil e eficiente, Bootstrap adapta seus projetos web com um simples código, de celulares para tablets e para desktops através de media queries CSS.

Muitos recursos

Com Bootstrap, você tem uma extensa e ótima documentação para elementos HTML comuns, dezenas de componentes CSS e HTML personalizados e incríveis plugins jQuery.

Bootstrap tem código aberto. É hospedado, desenvolvido e gerenciado no GitHub.
